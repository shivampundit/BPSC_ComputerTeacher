## ðŸ’¡ Study Tips

### For Better Retention
1. **Write by hand:** Write important algorithms
2. **Teach someone:** Explain concepts to others
3. **Draw diagrams:** Visualize data structures
4. **Create mnemonics:** For remembering complexities
5. **Space repetition:** Review after 1, 3, 7 days

### For Problem Solving
1. **Pattern recognition:** Identify common patterns
2. **Start simple:** Solve with small examples
3. **Think edge cases:** Empty, single element, large numbers
4. **Trace manually:** Don't just read code
5. **Time yourself:** Practice under time pressure

### For MCQs
1. **Read carefully:** Don't rush
2. **Eliminate options:** Remove obviously wrong answers
3. **Check all options:** Don't stop at first "correct" one
4. **Look for keywords:** "always," "never," "must," "may"
5. **Manage time:** 30-40 seconds per question
